# Admin Panel Project

This is a **Food Delivery Admin Panel** with separate **client** and **server** folders.
It allows admins to manage users, products, categories, and orders.

## Features
- Dashboard showing total users, products, orders, and revenue
- User Management (Add, Update, Delete, List)
- Category Management
- Product Management
- Order Creation

## Technologies
- React, Tailwind CSS
- Node.js, Express, MongoDB, Mongoose
- JWT Authentication with Cookies
